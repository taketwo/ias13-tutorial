Requirements
------------

  * PCL 1.7.2 (git master)
  * Boost 1.53 or above
  * Eigen 3.2 or above
  * Qt, QVTK

Installation
------------

    git clone https://github.com/taketwo/ias13-tutorial.git
    cd ias13-tutorial
    mkdir build
    cd build
    cmake ..
    make -j2

Usage
-----

From `build` directory:

    ./tutorial ../data/test1.pcd
